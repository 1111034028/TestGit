# 專案頁面功能說明書

本文件詳細描述專案中各個網頁的功能與用途，分為 **HW (課堂作業)** 與 **Final (音樂串流平台期末專案)** 兩大區塊。

---

## 📂 HW (課堂作業區)

此區塊主要展示基礎資料庫操作 (CRUD) 與相簿功能。

### 1. 作業首頁

- **檔案**: `hw/index.php`
- **功能**:
  - 作為作業區的入口網頁。
  - 顯示臺中科大圖片，並透過導覽列 (Navigation Bar)連結至通訊錄管理、相簿等功能。

### 2. 通訊錄管理 (學生列表)

- **檔案**: `hw/contacts.php`
- **功能**:
  - **資料列表**: 以表格分頁顯示所有學生資料 (學號、姓名、電話、地址、Email 等)。
  - **分頁功能**: 支援資料分頁瀏覽，避免單頁資料過多。
  - **操作入口**: 提供「新增資料」、「編輯」、「刪除」以及「查詢」功能的連結。

### 3. 新增通訊錄

- **檔案**: `hw/add.php`
- **功能**:
  - 提供表單供使用者輸入新的學生資料。
  - 包含學號重複檢查與基本資料驗證。
  - 提交後將資料寫入資料庫 `students` 資料表。

### 4. 編輯通訊錄

- **檔案**: `hw/edit.php` (以及 `hw/update.php`)
- **功能**:
  - 根據選擇的學生 (學號)，自動帶出目前的資料供修改。
  - 允許更新姓名、地址、電話等級資料，但通常不允許修改主鍵 (學號)。

### 5. 搜尋通訊錄

- **檔案**: `hw/search.php`
- **功能**:
  - 提供關鍵字搜尋欄位。
  - 依據姓名或學號篩選通訊錄資料，從大量資料中快速找到特定學生。

### 6. 相簿瀏覽

- **檔案**: `hw/album.php`
- **功能**:
  - **照片網格**: 以卡片式網格展示所有上傳的照片。
  - **燈箱效果 (Lightbox)**: 點擊照片可放大瀏覽，無需離開頁面。
  - **管理**: 顯示照片標題與上傳日期。

### 7. 相片管理 (CRUD)

- **檔案**: `hw/albumAdd.php`
  - **功能**: 提供上傳介面，允許選擇圖片檔案並填寫標題後上傳至伺服器。
- **檔案**: `hw/albumEdit.php`, `hw/albumEditAct.php`
  - **功能**: 編輯已上傳照片的資訊（如標題）並寫入資料庫。
- **檔案**: `hw/albumDelete.php`
  - **功能**: 刪除指定的照片記錄。

### 8. SDGs 永續發展目標列表

- **檔案**: `hw/itemList.php`
  - **功能**: (靜態版) 使用 PHP 陣列 (`array`) 儲存 SDGs 圖片、標題與描述，並透過迴圈與自訂函式 `printItem()` 輸出 HTML 結構。
- **檔案**: `hw/itemList_1.php`
  - **功能**: (資料庫版) 功能同上，但資料改為從 MySQL 資料庫的 `sdgs` 資料表中讀取，展示資料庫連線與查詢的應用。

### 9. 使用者註冊 (表單處理練習)

- **檔案**: `hw/getForm.php`
  - **功能**:
    - 使用者註冊表單頁面。
    - 包含 Cookie 讀取功能，若使用者曾填寫過，會嘗試自動帶入欄位值。
- **檔案**: `hw/getData.php`
  - **功能**:
    - 處理 `getForm.php` 送出的資料。
    - 執行重複性檢查 (姓名/地址/生日/帳密完全相同時不重複註冊)。
    - 自動產生學號 (S00X) 並將新使用者寫入資料庫。

### 10. 貸款本金和試算

- **檔案**: `hw/input.php`
  - **功能**: 提供填寫貸款金額、年利率、還款年數的表單。
- **檔案**: `hw/load.php`
  - **功能**: 接收表單數據，計算年複利後的本利和及還款總金額，並顯示結果。

### 11. 影像濾鏡處理

- **檔案**: `hw/photo.php`
  - **功能**:
    - 允許上傳圖片並調整影像參數 (亮度、對比、灰階、褐色調、負片)。
    - 使用 CSS `filter` 屬性即時預覽濾鏡效果。
    - 結合檔案上傳與表單參數傳遞的綜合練習。

### 12. 貓咪資料表單 (表格排版)

- **檔案**: `hw/tableForm.php`
  - **功能**: 展示複雜的 HTML 表格 (`table`) 與表單元素 (下拉選單、單選框、核取方塊) 的排版應用。
- **檔案**: `hw/showData.php`
  - **功能**: 接收並顯示 `tableForm.php` 提交的詳細資料，包含圖片上傳的後端處理與顯示。

## 🎵 Final (音樂串流平台期末專案)

此區塊為一個功能完整的現代化音樂串流 Web App。

### 1. 音樂播放主頁 (探索)

- **檔案**: `final/index.php`
- **功能**:
  - **核心首頁**: 使用者登入後的主要介面。
  - **最新發行**: 展示最新上傳的單曲卡片，支援懸停播放效果。
  - **全域播放器**: 位於畫面底部的固定式播放器，包含播放/暫停、上一首/下一首/重複播放、音量控制及進度條。
  - **自動播放**: 點選歌曲後自動載入並播放，支援佇列清單。

### 2. 會員驗證

- **檔案**: `final/login.php`
  - **功能**: 使用者登入介面，包含帳號密碼驗證與 Google 快速登入 (OAuth) 按鈕。
- **檔案**: `final/register.php`
  - **功能**: 新用戶註冊頁面，收集學號、姓名、帳號、密碼等資訊以建立會員資料。

### 3. 創作者工作室 (Creator Studio)

- **檔案**: `final/creator.php`
- **功能**:
  - **儀表板**: 專為創作者設計的後台。
  - **作品管理**: 列出該使用者上傳的所有歌曲。
  - **數據統計**: 顯示歌曲的播放次數累計。
  - **操作**: 提供「上傳新歌」、「編輯歌曲資訊」與「刪除歌曲」的快速入口。

### 4. 上傳音樂

- **檔案**: `final/upload.php`
- **功能**:
  - 允許創作者上傳 MP3 音訊檔與封面圖片。
  - 自動偵測音訊長度 (Duration)。
  - 填寫歌曲標題、描述、歌詞與類別。

### 5. 歌曲編輯

- **檔案**: `final/song_edit.php`
- **功能**:
  - 修改已上傳歌曲的 metadata（如標題、描述、歌詞）。
  - Updates existing records strictly checking permissions (only Owner or Admin can edit).

### 6. 搜尋系統

- **檔案**: `final/search.php`
- **功能**:
  - **即時搜尋**: 根據關鍵字搜尋歌曲標題、藝人名稱或專輯。
  - **結果播放**: 搜尋結果直接整合播放按鈕，點擊即可在底部播放器播放。
  - **無縫體驗**: 搜尋時不會中斷當前正在播放的音樂（透過 AJAX 載入或單頁應用設計概念）。

### 7. 使用者個人檔案

- **檔案**: `final/profile.php`
- **功能**:
  - **個人資料**: 顯示並允許編輯使用者暱稱、頭像。
  - **密碼管理**: 提供修改登入密碼的功能。

### 8. 我的歌單

- **檔案**: `final/my_playlists.php`
- **功能**:
  - 顯示使用者建立的所有播放清單。
  - 點擊清單可進入檢視該清單內的歌曲並進行播放。

### 9. 系統管理員後台

- **檔案**: `final/admin.php` & `final/admin_users.php`
- **功能**:
  - **權限控管**: 僅限 Admin 角色進入。
  - **全站統計**: 檢視平台總歌曲數、總用戶數。
  - **內容審查**: 強制下架違規歌曲。
  - **用戶管理**: 檢視所有註冊用戶，並可進行權限變更（如將一般用戶升級為管理員）。

### 10. 資訊頁面

- **檔案**: `about.php`, `privacy.php`, `terms.php`, `contact_us.php`
- **功能**:
  - 提供平台關於我們、隱私權政策、服務條款與聯絡客服的靜態資訊頁面，採用現代化排版設計。
